README

Front end for the project using React JS, Bootstrap, Axios, Speakeasy and some vanilla JavaScript components.
App.js is the home page and first file to load. Index.js and ServiceWorker.js contain the setup for React,
index.css is the css sheet corresponding to index.js.
SearchResults.js and SignUp.js are separate webpages.
Axios is used to communicate with backend via GET and POST requests
Speakeasy is used for dual authentication process via node.js
